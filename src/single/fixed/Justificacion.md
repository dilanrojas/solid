# Justificación
Se crearon las clases UserLogin y UserRegister para administrar las tareas de registro e ingreso. También se creó la clase CredentialsValidator con el propósito de comprobar las credenciales proporcionadas por el usuario (En este caso utilizando valores "quemados" como prueba).

La clase UserDataManager se encarga únicamente de simular (en este caso) el almacenamiento de las credenciales, que luego son solicitadas para el ingreso (login).
